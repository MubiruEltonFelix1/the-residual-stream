# 02 — Transformer

**Status:** Planned / Not yet built

This folder will implement the transformer architecture from *Attention Is All You Need* (Vaswani et al. 2017):

- Token embeddings + positional encoding
- Multi-head self-attention
- Residual connections and layer norm
- Feed-forward blocks
- Causal masking for autoregressive generation

Once built, this folder will connect the BPE tokenizer (`01`) with the training loop (`03`).
