# 05 — Scaling Laws

**Status:** Planned / Not yet built

This folder will explore compute-optimal training from *Training Compute-Optimal Large Language Models* (Hoffmann et al. 2022) and related scaling-law work:

- Chinchilla optimal ratio for parameter count vs data size
- Compute budget allocation
- Scaling trends for loss vs model size
- The training-compute frontier

The code here will be analytical and simulation-heavy rather than a full training run.
