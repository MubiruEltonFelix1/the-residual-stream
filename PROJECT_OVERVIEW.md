# Project Overview

## 1. Purpose
This repository is my public build log for learning and implementing core machine learning ideas from first principles.

## 2. Scope
The project covers foundations, paper implementations, experiments, paper notes, weekly builds, and supporting lecture/exercise material.

## 3. Structure
- foundations: linear algebra, calculus, and probability
- implementations: tokenizer, transformer, training loop, LoRA, and scaling-law work
- experiments: hypothesis-driven studies and ablations
- paper_notes: structured reading notes on papers
- lectures_exercises: lecture material and notebook practice
- research_papers: source papers and PDFs
- weekly_builds: weekly integrated outputs

## 4. Current Status
The repository is in active build mode. The foundations and documentation structure are in place, and the implementation folders are being prepared for code and notes.

## 5. Working Rules
- Derive ideas before relying on them.
- Implement the paper, not the shortcut.
- Keep code inspectable and testable.
- Record confusion as well as results.

## 6. Technical Direction
The code path is centered on JAX, with a focus on transparent implementation rather than framework-heavy abstractions.

## 7. Risks and Constraints
The main risks are overcomplicating early implementations, losing mathematical clarity, and drifting away from the original paper methods.

## 8. Next Steps
- Add starter READMEs and notes inside each implementation subfolder.
- Keep linking the theory folders to runnable code.
- Use experiments to validate assumptions and surface weak spots.
